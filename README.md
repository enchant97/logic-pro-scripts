# Logic Pro Scripts
A collection of scripts for Logic Pro's "Scripter" effect.

## Scripts
### Poly Aftertouch to MPE
Converts Polyphonic Aftertouch to MPE Channel Pressure.

Useful for when a MIDI controller supports Polyphonic Aftertouch but not MPE, allowing for easier editing of recorded MIDI data in piano roll.

Usage:

1. Add Scripter to top of MIDI effects for instrument
2. Right-Click on Scripter and enable "Record MIDI to Track Here"
3. Make sure MIDI controller is set to Poly Aftertouch
4. Record as normal
